## 🛠️ **Installation**

### 📋 **Prerequisites**

Before you begin, ensure you have the following installed:

- 🐍 **Python 3.8+**
- 📦 **pip** (Python package manager)
- 🌐 **Git**
- 🔧 **Virtual Environment** (recommended)

### 🚀 **Quick Install**

```bash
# 1️⃣ Clone the repository
git clone https://github.com/reo-sudo/Retro-Book-library.git
cd Retro-Book-library

# 2️⃣ Create virtual environment
python -m venv retro_env
source retro_env/bin/activate  # On Windows: retro_env\Scripts\activate

# 3️⃣ Install dependencies
pip install -r requirements.txt

# 4️⃣ Setup database
python manage.py migrate

# 5️⃣ Create superuser (optional)
python manage.py createsuperuser

# 6️⃣ Run development server
python manage.py runserver
```

### 🔧 **Environment Variables**

Create a `.env` file in the root directory:

```env
# Django Settings
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1

# Google Books API
GOOGLE_BOOKS_API_KEY=your-google-books-api-key

# Database (Optional - defaults to SQLite)
DATABASE_URL=sqlite:///db.sqlite3

# Email Settings (Optional)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=your-email@gmail.com
EMAIL_HOST_PASSWORD=your-app-password
```

## 📚 **API Reference**

### 🔍 **Search Endpoints**

```python
# Search books
GET /api/books/search/?q=python&limit=10

# Get book details
GET /api/books/{book_id}/

# Add book to library
POST /api/library/add/
{
    "book_id": "abc123",
    "user_id": 1
}
```

### 👤 **User Endpoints**

```python
# User profile
GET /api/user/profile/

# User library
GET /api/user/library/

# Update profile
PUT /api/user/profile/
{
    "username": "newname",
    "email": "new@email.com"
}
```

## 🎨 **Customization**

### 🌈 **Theming**

Create custom themes by modifying `static/css/retro.css`:

```css
/* Custom Neon Theme */
.neon-theme {
    --primary-color: #FF6B6B;
    --secondary-color: #4ECDC4;
    --accent-color: #DDA0DD;
    --text-color: #FFFFFF;
    --bg-color: #1a1a1a;
}

/* Glow Effects */
.neon-glow {
    box-shadow: 0 0 20px var(--primary-color);
    animation: pulse 2s infinite;
}

@keyframes pulse {
    0% { opacity: 1; }
    50% { opacity: 0.7; }
    100% { opacity: 1; }
}
```

### 🎭 **Custom Components**

```html
<!-- Custom Book Card -->
<div class="retro-card">
    <div class="card-header neon-glow">
        <h3>{{ book.title }}</h3>
    </div>
    <div class="card-body">
        <p>{{ book.description|truncatewords:20 }}</p>
        <div class="card-actions">
            <button class="btn-neon">Add to Library</button>
        </div>
    </div>
</div>
```
